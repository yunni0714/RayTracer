RayTracer — 디자인 시안

같은 UI 조각(① 5×5 보드 + 레이저, ② 팔레트 패널, ③ 라이브러리 맵카드)을 세 가지 비주얼 방향으로 렌더했어. 실제 기물 SVG 그대로 사용. 색·타이포·버튼 상태·radius 토큰은 각 방향 아래에 같이 표기 — 고른 게 바로 디자인 토큰이 됨.

→ 한 방향을 고르거나, "보드는 다크 / 카드는 클린" 식으로 섞어서 알려줘. 세부(색 하나 바꿔 등)도 환영.

A · 클린 라이트 뉴트럴 + 포인트색. 차분·정돈, 가독성 우선. 현재 톤의 정제 버전.
B · 다크 테크 / 네온 레이저 어두운 배경 + 네온 빔 글로우. 레이저 정체성 극대화, 대비 강함.
C · 플레이풀 소프트 둥근 radius·생동 색·부드러운 그림자. 가벼운 퍼즐게임 느낌.
